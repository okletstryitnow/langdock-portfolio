# FlowSync Product Overview

## Executive Summary

FlowSync is an all-in-one workflow automation and project management platform designed for modern, fast-growing teams. Built by a team of ex-Asana and ex-Zalando engineers, FlowSync combines powerful project management capabilities with seamless time tracking, team communication, and deep integrations with the tools your team already uses.

**Key Differentiators:**
- Native time tracking built into every task
- Bidirectional sync with Jira, GitHub, and Slack
- DACH-focused with German-language support and GDPR compliance
- Competitive pricing for mid-market companies

---

## Core Features

### 1. Project Management

FlowSync provides flexible project management that adapts to your team's workflow.

**Kanban Boards**
- Drag & drop cards across customizable columns
- WIP limits to prevent bottlenecks
- Swimlanes for priority grouping
- Quick filters by assignee, due date, or label

**Timeline View**
- Gantt-style visualization of project schedules
- Dependency linking between tasks
- Critical path highlighting
- Export to PDF for stakeholder presentations

**List View**
- Spreadsheet-like task management
- Bulk editing capabilities
- Custom fields (text, number, date, dropdown)
- Group by any field

**Templates**
- Pre-built templates for Agile Sprints, Marketing Campaigns, Client Projects
- Custom template creation
- Template sharing across workspace

### 2. Time Tracking

Integrated time tracking eliminates the need for separate tools.

**One-Click Timer**
- Start/stop timer directly from any task
- Browser extension for quick access
- Mobile app support (iOS/Android)
- Automatic idle detection

**Manual Time Entry**
- Add time entries retroactively
- Round to nearest 15 minutes (configurable)
- Add notes to time entries
- Split entries across multiple tasks

**Reporting**
- Weekly/monthly time reports per user
- Project-based time summaries
- Billable vs. non-billable breakdown
- Export to CSV, Excel, or PDF

**Billing Integration**
- Hourly rate per user or project
- Invoice generation (Pro tier)
- Integration with accounting tools (Zapier)

### 3. Team Communication

Keep conversations in context, not scattered across email and chat.

**Task Comments**
- Threaded discussions on every task
- Rich text formatting (Markdown)
- File attachments (up to 25MB)
- @mention teammates for notifications

**Real-Time Updates**
- Live activity feed
- Typing indicators
- Instant notification delivery
- Presence indicators (online/away/DND)

**Slack Integration**
- Receive task notifications in Slack
- Create tasks from Slack messages
- /flowsync commands for quick actions
- Thread sync for extended discussions

### 4. Integrations & API

Connect FlowSync to your existing tool stack.

**Native Integrations**
- Slack (notifications, commands, thread sync)
- Jira (bidirectional sync, status mapping)
- GitHub (commit tracking, auto-linking PRs)
- Google Calendar (deadline sync, time blocking)
- Zapier (1000+ app connections)

**REST API**
- Full CRUD operations on all resources
- OAuth 2.0 and API key authentication
- Webhooks for real-time events
- Rate limits: 1000/hour (Pro), 5000/hour (Enterprise)

**Enterprise Features**
- SAML 2.0 SSO (Okta, Azure AD, OneLogin)
- SCIM provisioning
- Custom domain
- Dedicated IP whitelisting
